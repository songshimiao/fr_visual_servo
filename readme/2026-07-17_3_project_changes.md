## 2. 系统参数优化

### 静止检测阈值调整
- `--settle-pos-thresh` 默认值从 0.003m 调整为 0.0008m (8mm → 0.8mm)
- `--settle-ang-thresh` 默认值从 2.0° 调整为 0.6° (2° → 0.6°)
- 更严格的静止检测阈值，提高系统稳定性

### 安全退出动作优化
- 机器人安全退出时的归零动作调整
- 从 `[90, -90, 90, -90, -90, -90]` 调整为 `[90, -90, 90, -90, -90, 0]`
- 避免某些关节位置可能带来的安全隐患

## 3. 文件结构调整

### .gitignore 更新
- 新增 `scripts/__pycache__` 到忽略列表

## 4. 改动影响

### 正面影响
- 提高了视觉伺服系统的精度和稳定性
- 更严格的静止检测条件减少了误判
